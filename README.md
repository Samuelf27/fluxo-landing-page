## 📌 Sobre o projeto

**Fluxo** é uma *landing page* de demonstração para um produto fictício de gestão financeira. O objetivo é exercitar **design de produto e front-end moderno**: hero com mockup de dashboard, recursos, passo a passo, tabela de planos e CTA — tudo com tema dark e animações sutis.

## ✨ Destaques

- 🪟 Mockup de dashboard com gráfico de barras animado
- 💳 Tabela de planos (Free / Pro / Família) com destaque de plano popular
- 🎞️ Animações de *reveal* ao rolar
- 📱 Layout 100% responsivo
- ⚡ Estático e leve, sem frameworks

## 🛠️ Tecnologias

HTML5 · CSS3 · JavaScript (vanilla)

## 🚀 Como rodar

```bash
git clone https://github.com/Samuelf27/fluxo-landing-page.git
cd fluxo-landing-page
# abra index.html no navegador (ou use Live Server)
```

> 💡 **Fluxo** é um produto fictício, criado apenas para fins de demonstração.
